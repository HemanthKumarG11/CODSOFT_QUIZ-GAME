# Python Quiz Game

![Quiz Game](quiz_game.png)

This is a simple Python quiz game that asks users multiple-choice questions on various topics. It provides feedback on correct/incorrect answers and calculates the final score.

## Features

- Interactive GUI-based quiz game.
- Multiple-choice questions with randomized options.
- Feedback on each question.
- Display of the final score and performance.

## Technologies Used

- Python
- Tkinter (for the graphical user interface)
